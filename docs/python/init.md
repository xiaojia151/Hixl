## 函数功能<a name="zh-cn_topic_0000001481404214_zh-cn_topic_0000001488949573_zh-cn_topic_0000001357384997_zh-cn_topic_0000001312399929_section3870635"></a>

初始化LLMDataDist，需要在init的options中配置CacheManager模式，即[enable\_cache\_manager](enable_cache_manager.md)设置为True或者指定[local\_comm\_res](local_comm_res.md)。

该模式下，Decode和Prompt可以双向拉取Cache。

## 调用示例<a name="section17821439839"></a>

```
from llm_datadist import LLMDataDist, LLMRole, LLMConfig
llm_datadist = LLMDataDist(LLMRole.PROMPT, 0)
llm_config = LLMConfig()
llm_config.enable_cache_manager = True
llm_config.device_id = 0
engine_options = llm_config.generate_options()
llm_datadist.init(engine_options)
```

## 返回值<a name="zh-cn_topic_0000001481404214_zh-cn_topic_0000001488949573_zh-cn_topic_0000001357384997_zh-cn_topic_0000001312399929_section45086037"></a>

正常情况下无返回值。

异常情况会抛出[LLMException](LLMException.md)。

参数错误可能抛出TypeError或ValueError。

## 约束说明<a name="section1391611144104"></a>

初始化成功后，系统退出前需要调用[finalize](finalize.md)。
