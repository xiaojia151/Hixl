```
llm_config = LLMConfig()
llm_config.local_comm_res = '''{
    "status": "completed",
    "version": "1.0",
    "server_list": [
        {
            "server_id": "node_0",
            "device": [
                {
                    "device_id": "0",
                    "device_ip": "x.x.x.x"
                }
            ]
        }
    ]
}'''
```

## 返回值<a name="zh-cn_topic_0000001481404214_zh-cn_topic_0000001488949573_zh-cn_topic_0000001357384997_zh-cn_topic_0000001312399929_section45086037"></a>

正常情况下无返回值。

参数错误可能抛出TypeError或ValueError。

## 约束说明<a name="section2411483221"></a>

配置了该option后，存在如下约束。

-   调用[link\_clusters](link_clusters.md)或[unlink\_clusters](unlink_clusters.md)时，如果local\_comm\_res不为空，则不需要配置“clusters”参数中的“append\_local\_ip\_info”信息。如果local\_comm\_res为空，则需要配置“clusters”参数中的“append\_local\_ip\_info”信息。
-   当前不支持[enable\_cache\_manager](enable_cache_manager.md)和[enable\_remote\_cache\_accessible](enable_remote_cache_accessible.md)配置为“False”的场景。
